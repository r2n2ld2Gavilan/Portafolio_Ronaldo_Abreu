# 🚀 Portafolio Web Profesional - Ronaldo Abreu

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **portafolio web profesional** con estética **ciberpunk oscura**, diseño moderno, estructura responsiva y modo **Dark/Light**.

El portafolio fue creado como parte de una asignación académica utilizando tecnologías frontend básicas y modernas. Su objetivo principal es presentar el perfil profesional de **Ronaldo Abreu**, destacando su formación como estudiante de **Ingeniería en Sistemas** y su experiencia como **Mecánico Automotriz**.

## 🛠️ Tecnologías Utilizadas

El proyecto fue desarrollado utilizando:

* CSS3
* Bootstrap 5.3.3
* JavaScript
* Font Awesome
* Devicon
* EmailJS / Mailto
* Canvas API para fondo Matrix

---

## 🌌 Características Principales

El portafolio incluye las siguientes secciones:

* Navbar responsivo con navegación interna.
* Hero principal con nombre, profesión, biografía y foto de perfil.
* Fondo Matrix translúcido desarrollado con JavaScript y Canvas.
* Modo Dark/Light aplicado a toda la interfaz.
* Stack tecnológico con íconos oficiales.
* Sección de proyectos con filtros dinámicos.
* Experiencia laboral en tarjetas centradas.
* Videos destacados incrustados desde YouTube.
* Formulario de contacto moderno y responsivo.
* Footer con enlaces sociales.
* Botón “Volver Arriba”.
